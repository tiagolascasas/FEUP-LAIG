Class: PrimitiveCylinder

PrimitiveCylinder(scene, height, bottomR, topR, stacks, slices, topcap, botcap)

new PrimitiveCylinder(scene, height, bottomR, topR, stacks, slices, topcap, botcap)

Primitive that represents a cylinder with configurable dimensions and whose base is on the x0y plan
Parameters:
Name Type Description
scene CGFScene the scene to which this primitive will belong
height int the height of the cylinder
bottomR int the radius of the bottom lid
topR int the radius of the top lid
stacks int the number of stacks
slices int the number of slices
topcap boolean flag that indicates whether to include the top lid or not
botcap boolean flag that indicates whether to include the bottom lid or not
