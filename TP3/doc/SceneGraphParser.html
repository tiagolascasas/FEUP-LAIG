Class: SceneGraphParser

SceneGraphParser(filename, scene, initScene)

new SceneGraphParser(filename, scene, initScene)

SceneGraphParser class, which parses a xml file with a scene graph and builds an ObjectGraph with the nodes information, and also holds all the configuration and lights definitions
Parameters:
Name Type Description
filename String the name of the LSX file containing the graph information
scene CGFscene the scene to bind this graph to
initScene Boolean indicates whether the initials, illumination and lights information of the file should be applied to the scene or ignored
